## рҹҡҖ Getting Started

### Prerequisites
```
вң… Node.js 24+
вң… pnpm
вң… Neon DB account  вҶ’  neon.tech (free)
вң… Groq API key     вҶ’  console.groq.com (free)
```

### 1пёҸвғЈ Clone & Install
```bash
git clone https://github.com/yourusername/fitness-app.git
cd fitness-app
pnpm install
```

### 2пёҸвғЈ Environment Setup
```env
# .env in project root

DATABASE_URL=your_neon_connection_string
XAI_API_KEY=your_xai_api_key
AI_INTEGRATIONS_OPENAI_BASE_URL=https://api.x.ai/v1
AI_INTEGRATIONS_OPENAI_API_KEY=your_xai_api_key
```

### 3пёҸвғЈ Database Setup
```bash
pnpm --filter @workspace/db run push
```

### 4пёҸвғЈ Fire It Up рҹ”Ҙ
```bash
# Terminal 1 вҖ” API Server
pnpm --filter @workspace/api-server run dev

# Terminal 2 вҖ” Frontend
pnpm --filter @workspace/fitness-app run dev
```

```
Open вҶ’ http://localhost:5173 рҹҡҖ
```
